# General Notes

Types:

* data-controller
* data-target
* data-action

```
`click->` has been omitted from the action descriptor.
`data-action="click->clipboard#copy` vs `data-action="clipboard#copy`
```

Default Events:

| Element           | Default Event |
| ----------------- |:-------------:|
| a                 | click         |
| button            | click         |
| form              | submit        |
| input             | change        |
| input type=submit | click         |
| select            | change        |
| textarea          | change        |
